# xTRAI - AI Automation & Optimization Platform

Welcome to xTRAI, a modern web application built to empower businesses with intelligent AI automation, helping them scale faster, reduce operational costs, and streamline workflows.

## 🚀 Features
- **Intelligent Automation:** Automate repetitive tasks with AI-powered workflows.
- **AI-Powered Insights:** Leverage machine learning to gain actionable real-time data insights.
- **Modern & Responsive UI:** Built with Tailwind CSS, Radix UI, and Framer Motion for a stunning, accessible user experience.
- **Sleek Metallic Theme:** A unified and professional "metallic purple" aesthetic across all pages and interactions.

## 🛠️ Built With
- [Vite](https://vitejs.dev/) - Next Generation Frontend Tooling
- [React](https://reactjs.org/) - A JavaScript library for building user interfaces
- [TypeScript](https://www.typescriptlang.org/) - Typed JavaScript at Any Scale
- [Tailwind CSS](https://tailwindcss.com/) - Rapidly build modern websites without ever leaving your HTML
- [Framer Motion](https://www.framer.com/motion/) - Open source, production-ready animation and gesture library for React
- [shadcn/ui](https://ui.shadcn.com/) - Beautifully designed components that you can copy and paste into your apps

## 💻 Getting Started

To get a local copy up and running, follow these simple steps:

### Prerequisites
Make sure you have Node.js and npm installed on your local machine.

### Installation

1. Clone the repository
   ```sh
   git clone <YOUR_GIT_URL>
   ```

2. Navigate into the project directory
   ```sh
   cd <YOUR_PROJECT_NAME>
   ```

3. Install NPM packages
   ```sh
   npm install
   ```

4. Start the development server
   ```sh
   npm run dev
   ```

The application will be running at `http://localhost:8080/` (or another port depending on your Vite config).
